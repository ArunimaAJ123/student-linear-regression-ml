# Student Performance Prediction using Linear Regression

## Project Description
This project predicts student performance using a Linear Regression model.
The model achieved an accuracy (R² score) of 72%.

## Dataset
The dataset was taken from Kaggle and contains student-related data.

## Methodology
- Data loading using Pandas
- Data preprocessing
- Train-test split
- Linear Regression model training
- Model evaluation

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Result
Accuracy achieved: 72%

## Future Scope
The model performance can be improved using advanced machine learning algorithms.
